## Purpose

This improvement changes the daily report from a plain status reminder into an event-conclusion report. The report now groups recent project and LINE signals into concrete event cards with:

- Subject
- Target
- Impact
- Current conclusion
- Suggested next task

The goal is to make the daily report useful for management decisions: what happened, why it matters, what conclusion can be drawn, and what should be done next.
