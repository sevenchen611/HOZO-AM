# UPGRADE-2026-06-10-AM-IMP-2026.0610.02

Project: HOZO AM

Improvement: Task source evidence gate

## What Changed

- HOZO AM hourly LINE task reconciliation now requires source evidence before a
  new formal task can be created.
- Task creation must write the LINE conversation, meeting record, report clue,
  system suggestion trace, attachment, or linked source page into the task body
  or source/evidence fields.
- Task status, owner, due date, next-step, priority, or scope updates must append
  the source clue that caused the change.
- If source evidence cannot be attached, the item should remain candidate,
  pending confirmation, or logged for source recovery instead of becoming a
  confirmed formal task.

## Verification

- Project-local `config/hourly-line-task-reconciliation.json` includes
  `sourceEvidenceGate`.
- AMCore source-evidence validation tool can scan generated task pages.

Production Notion write-flow verification remains a project-local follow-up.
