Target project: HOZO AM

Owner: HOZO AM / Codex

## Purpose

This improvement makes responsibility assignment more precise. Instead of choosing a responsible owner from one large list, the workflow now narrows the candidate list in this order:

1. Select the project or workstream.
2. Show only LINE groups assigned to that project.
3. Select the main LINE group.
4. Show only known members of that group.
5. Select the main owner and optional backup owner.
6. Write the LINE target result fields for report and follow-up use.
